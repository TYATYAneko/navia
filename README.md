# Navia

中学生のための学習クイズサイト

## 概要

Naviaは中学生が5教科（国語・数学・英語・理科・社会）を楽しく学べるクイズ形式の学習サイトです。

## 機能

- **5教科対応**: 数学・国語・英語・理科・社会
- **単元別学習**: 各教科ごとに単元を選んで学習可能
- **クイズ形式**: 4択問題で理解度をチェック
- **答え合わせ機能**: 回答後に「答え合わせ」ボタンで正解を確認
- **解説表示**: 各問題に詳しい解説付き
- **スコア表示**: クイズ終了後に成績を表示
- **レスポンシブデザイン**: PC・スマホ両対応

## 教科と単元

| 教科 | 単元 |
|------|------|
| 数学 | 計算、方程式、図形、関数 |
| 国語 | 漢字、文法、語彙・表現 |
| 英語 | 単語、文法、会話・表現 |
| 理科 | 物理、化学、生物、地学 |
| 社会 | 地理、歴史、公民 |

## ファイル構成

```
Navia/
├── index.html          # トップページ（教科選択）
├── select-unit.html    # 単元選択ページ
├── quiz.html           # クイズページ
├── css/
│   └── style.css       # スタイルシート
├── js/
│   └── quiz.js         # クイズロジック
└── data/
    └── questions.js    # 問題データ
```

## 使い方

1. `index.html` をブラウザで開く
2. 学習したい教科を選択
3. 学習したい単元を選択（または「全単元」でランダム出題）
4. 問題に回答し「答え合わせ」ボタンで確認
5. 全問終了後、スコアを確認

## 問題の追加方法

`data/questions.js` を編集して問題を追加できます。

```javascript
{
    question: "問題文",
    options: ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
    answer: 0,  // 正解の選択肢番号（0から始まる）
    explanation: "解説文"
}
```
